# MAKE SUSHI!

## 概要
2017年制作

部内ハッカソンで作ってるはず(うろおぼえ)
一部プログラムとSUSHIのドット絵を担当
~~なぜかソースコードがない~~
多分自分が担当した部分のコードがあった

## ゲーム説明
大将を操作し正面の客にお寿司を提供しよう！
正面に客がいなかったり、間違ったお寿司を提供すると失敗！
制限時間内により多くのお寿司を提供して、スコアを伸ばそう！
