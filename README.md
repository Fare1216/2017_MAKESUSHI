# MAKE SUSHI!

## 概要
2017年制作

部内ハッカソンで作ってるはず(うろおぼえ)<br>
一部プログラムとSUSHIのドット絵を担当<br>
~~なぜかソースコードがない~~<br>
多分自分が担当した部分のコードがあった<br>

## ゲーム説明
大将を操作し正面の客にお寿司を提供しよう！<br>
正面に客がいなかったり、間違ったお寿司を提供すると失敗！<br>
制限時間内により多くのお寿司を提供して、スコアを伸ばそう！<br>

## 操作説明
### 大将の移動
J:左<br>
K:中央<br>
L:右<br>

### 寿司選択

S:マグロ<br>
